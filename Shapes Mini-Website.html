SQUARE

x - border lenght

AREA = x 2 = x • x
PERIMETER = 4x = x + x + x + x
RECTANGLE

x, y - borders lenght

AREA = xy
PERIMETER = 2x + 2y = x + x + y + y
CIRCLE

r - radius of circle
constant pi = 3,1415

AREA = pi • r 2
PERIMETER = 2 • pi • r
TRIANGLE

a, b, c - triangle sides lenght
alpha, beta, gama - angle
h - height

AREA = sin(gama) • (a • b) / 2
PERIMETER = a + b + c
SINUS THEORY -> sin(alpha) / a = sin(beta) / b = sin(gama) / c
PITAGOREAN THEORY -> a2 = b2 + c2
